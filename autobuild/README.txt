These scripts perform the autobuild fitting procedure.
Dependencies include phenix and ccp4 utilitiea, here accessed through sbgrid. 
Management of files, directoties, and job scheduling is up to the user.
autobuild_prep.sh -- prepares the necessary input files.
autobuild_default.sh --performs the full fitting procudure using defaults
autobuild_fixedmap.sh -- performs the fitting to a fixed map
autobuild_2FOFC.sh -- performs the fitting starting from the best fixed map result and makes new 2FOFC maps for fitting
gather.sh -- go through the files and gather information for ploting in excel
